# Gamlang documentation - mulint()

mulint(var, int)

| Parameter | Description |
| --------|--------|
| 1 | Integer variable |
| 2 | Integer |

## Description

Multiplies parameter 1 by parameter 2, and stores it in parameter 1.

## Code (see description)

```
mov al, [ptr]
mov ah, <val>
mul ah
mov [ptr],ax
```
